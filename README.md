# Java Task Manager (Console App)

This is a simple console-based Task Manager that allows users to:
- Add tasks
- Edit task status
- Delete tasks
- View all tasks

## How to Run
Compile and run the `Main.java` using any Java compiler.

## Features
- Object-Oriented Design
- Basic file handling (optional)
